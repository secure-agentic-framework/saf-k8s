# NIST SP 800-53 Revision 5 5.2.0 - SC-28

- Framework Code: NIST-800-53R5
- Requirement Title: Protection of Information at Rest

## Mapping Notes

Protecting agent data at rest (models, context, memory, credentials) prevents unauthorized access

1.2 - etcd encryption at rest; 7.1 - PV encryption; 4.3 - Secrets encryption at rest (KMS providers)
