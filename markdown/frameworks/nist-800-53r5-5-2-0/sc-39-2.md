# NIST SP 800-53 Revision 5 5.2.0 - SC-39(2)

- Framework Code: NIST-800-53R5
- Requirement Title: Separate Execution Domain Per Thread

## Mapping Notes

Separate execution domains per agent thread prevent cross-thread data leakage

8.1 - MIG partitioning (separate execution domains per GPU instance); 2.2 - RuntimeClass (separate execution domains via gVisor/Kata)
