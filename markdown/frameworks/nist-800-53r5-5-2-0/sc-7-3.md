# NIST SP 800-53 Revision 5 5.2.0 - SC-7(3)

- Framework Code: NIST-800-53R5
- Requirement Title: Access Points

## Mapping Notes

Limiting network access points to agent systems reduces attack surface

5.3 - Harden ingress (managed access points); 5.5 - Protect API server and service exposure (LoadBalancer, NodePort, ExternalIP restriction)
