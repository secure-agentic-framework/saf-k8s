- Framework Code: NIST-800-53R5
- Requirement Title: Block Communication from Non-organizationally Configured Hosts

## Mapping Notes

Blocking unauthorized hosts from communicating with agents prevents rogue connections

5.1 - Network policies (blocking communication from unauthorized pods); 2.5 - Node-level firewall
