# NIST SP 800-53 Revision 5 5.2.0 - SC-34(1)

- Framework Code: NIST-800-53R5
- Requirement Title: No Writable Storage

## Mapping Notes

Read-only agent containers prevent persistent compromise through filesystem immutability

3.2 - readOnlyRootFilesystem; 9.2 - Read-only model mounts for inference servers (no writable storage)
