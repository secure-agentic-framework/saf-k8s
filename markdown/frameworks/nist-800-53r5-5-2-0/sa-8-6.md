# NIST SP 800-53 Revision 5 5.2.0 - SA-8(6)

- Framework Code: NIST-800-53R5
- Requirement Title: Minimized Sharing

## Mapping Notes

Minimizing shared state between agents reduces cross-contamination and lateral movement risk

8.1 - MIG GPU partitioning (minimized hardware sharing); 7.2 - Namespace isolation (minimized resource sharing)
