# NIST SP 800-53 Revision 5 5.2.0 - SI-22

- Framework Code: NIST-800-53R5
- Requirement Title: Information Diversity

## Mapping Notes

Information diversity in agent inputs enables cross-validation and reduces manipulation risk

9.6 - Data integrity at scale (diversity in data sources for resilience); 9.1 - Federated learning (diverse participant data)
