# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PS.3.1

- Framework Code: NIST-SSDF
- Requirement Title: Securely archive the necessary files and supporting data to be retained for each software release.

## Mapping Notes

SAF-K8S requires retention policies for container images and artifacts (SAF-K8S-0601-005), model registry hardening with version retention (SAF-K8S-0905-003), artifact lifecycle management (SAF-K8S-0905-006), and encryption at rest for stored artifacts. Retention policies ensure integrity metadata is preserved alongside the artifacts it describes.
