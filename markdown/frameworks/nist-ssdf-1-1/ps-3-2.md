# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PS.3.2

- Framework Code: NIST-SSDF
- Requirement Title: Collect, safeguard, maintain, and share provenance data for all components of each software release (e.g., in an SBOM).

## Mapping Notes

SAF-K8S has dedicated controls for SBOM generation, ML-BOM generation, SBOM distribution as OCI artifacts, VEX publication, and SLSA build provenance. These directly implement provenance data collection and sharing for Kubernetes-deployed software.
