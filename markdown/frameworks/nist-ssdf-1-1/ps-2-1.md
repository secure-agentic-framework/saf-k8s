# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PS.2.1

- Framework Code: NIST-SSDF
- Requirement Title: Make software integrity verification information available to software acquirers.

## Mapping Notes

SAF-K8S requires image signing (cosign/Sigstore, Notary v2), SLSA provenance, and transparency logging to provide integrity verification for container images, ML models, and deployment artifacts.
