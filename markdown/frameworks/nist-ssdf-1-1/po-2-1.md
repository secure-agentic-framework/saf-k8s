# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PO.2.1

- Framework Code: NIST-SSDF
- Requirement Title: Create new roles and alter responsibilities for existing roles as needed to encompass all parts of the SDLC.

## Mapping Notes

SAF-K8S defines distinct roles (platform engineer, security engineer, ML engineer) with RBAC enforcement and separation of duties. While not SDLC-scoped, SAF-K8S operationalizes role separation for the infrastructure layer.
