# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PO.3.3

- Framework Code: NIST-SSDF
- Requirement Title: Configure tools to generate artifacts of their support of secure software development practices as defined by the organization.

## Mapping Notes

SAF-K8S requires generation of SBOMs, ML-BOMs, build attestations, and SLSA provenance as evidence artifacts of secure development practices.
