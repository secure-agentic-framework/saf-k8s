# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - RV.1.3

- Framework Code: NIST-SSDF
- Requirement Title: Have a policy that addresses vulnerability disclosure and remediation, and implement the roles, responsibilities, and processes needed to support that policy.

## Mapping Notes

SAF-K8S-1005-007 requires a vulnerability disclosure policy for Kubernetes infrastructure with defined intake channels, triage SLAs, roles and responsibilities, and coordination processes for multi-party vulnerabilities. Combined with IR lifecycle controls, this provides structured vulnerability disclosure and remediation at the infrastructure level.
