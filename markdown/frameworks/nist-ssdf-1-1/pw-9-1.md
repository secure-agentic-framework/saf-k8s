# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PW.9.1

- Framework Code: NIST-SSDF
- Requirement Title: Define a secure baseline by determining how to configure each setting that has an effect on security.

## Mapping Notes

SAF-K8S defines secure baseline configurations for Kubernetes components: CIS Benchmark automated assessment, Pod Security Standards, Cloud Native 8 secure defaults, and inference server hardening.
