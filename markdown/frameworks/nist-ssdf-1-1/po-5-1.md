# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PO.5.1

- Framework Code: NIST-SSDF
- Requirement Title: Separate and protect each environment involved in software development.

## Mapping Notes

SAF-K8S enforces environment separation between development, staging, and production through namespace isolation, network policies, and distinct cluster boundaries. AI-specific controls enforce separation between training, inference, and pipeline environments.
