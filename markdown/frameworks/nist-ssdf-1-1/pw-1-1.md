# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PW.1.1

- Framework Code: NIST-SSDF
- Requirement Title: Use forms of risk modeling -- such as threat modeling, attack modeling, or attack surface mapping -- to help assess the security risk.

## Mapping Notes

SAF-K8S includes dedicated threat modeling controls: STRIDE for Kubernetes AI systems, OCTAVE risk assessment, and MITRE ATT&CK for Containers mapping. These apply to the infrastructure platform, not to application software design.
