# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PO.1.1

- Framework Code: NIST-SSDF
- Requirement Title: Identify and document all security requirements for the organization's software development infrastructures and processes, and maintain the requirements over time.

## Mapping Notes

SAF-K8S controls define security requirements for the Kubernetes platform infrastructure that hosts AI workloads. The framing document, control skeleton, and compliance crosswalks serve as documented security requirements for the infrastructure layer.
