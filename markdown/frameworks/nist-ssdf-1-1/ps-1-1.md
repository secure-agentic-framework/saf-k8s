# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PS.1.1

- Framework Code: NIST-SSDF
- Requirement Title: Store all forms of code -- including source code, executable code, and configuration-as-code -- based on the principle of least privilege.

## Mapping Notes

SAF-K8S enforces least-privilege access to container images in registries, GitOps repositories, Helm charts, and Kubernetes manifests. Registry access controls, RBAC, and repository security controls restrict who can read and modify code and configuration artifacts.
