# NIST SP 800-218: Secure Software Development Framework (SSDF) Version 1.1 1.1 - PO.1.3

- Framework Code: NIST-SSDF
- Requirement Title: Communicate requirements to all third parties who will provide commercial software components to the organization for reuse.

## Mapping Notes

SAF-K8S supply chain controls enforce requirements on third-party components (container images, Helm charts, ML models) admitted to the cluster. SAF-K8S-0604-007 requires documented security requirements for third-party components and their communication to vendors. Admission policies, signature verification, and SBOM requirements operationalize these requirements at the infrastructure boundary.
