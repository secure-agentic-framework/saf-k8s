# SAF-K8S-0803-002 - InfiniBand and RoCE fabric security controls

- Domain: D08 - GPU, Accelerator, and Confidential Computing
- Knowledge Area: 8.3 - High-Performance Interconnect Security
- Maturity: Advanced
- Classification: ai-specific

## Statement

InfiniBand and RoCE (RDMA over Converged Ethernet) fabrics used for distributed AI training must enforce subnet manager access controls, partition key (P_Key) isolation between tenants, and network fabric segmentation to prevent unauthorized cross-partition communication.

## Objective

Secure the high-performance networking fabric that carries distributed training traffic (gradient synchronization, parameter updates) to prevent unauthorized access, traffic interception, and cross-tenant communication on shared InfiniBand or RoCE infrastructure.

## Rationale

InfiniBand and RoCE fabrics bypass traditional IP-based network security controls, operating at Layer 2 with direct memory access semantics. The subnet manager controls all fabric routing and access policies; unauthorized subnet manager access enables complete traffic redirection. Without partition key enforcement, any node on the fabric can communicate with any other node, enabling training data interception or gradient poisoning attacks across tenant boundaries.

## Scope

Applies to Kubernetes clusters using InfiniBand (ConnectX adapters, HDR/NDR switches) or RoCE for distributed AI training workloads. Includes subnet manager nodes, fabric switches, and host channel adapters. Ethernet-only clusters not using RDMA are not in scope. Managed cloud HPC networking must verify provider partition key and fabric isolation controls.

## Enforcement

- Primary Enforcement Point: InfiniBand subnet manager configuration, partition key tables on switches and HCAs, Kubernetes RDMA device plugin configuration, and network fabric management plane access controls.
- Implementation Surface: InfiniBand subnet manager (OpenSM or vendor SM) access controls and configuration, partition key (P_Key) tables on InfiniBand switches and Host Channel Adapters (HCAs), SR-IOV network operator and RDMA device plugin P_Key configuration per pod, InfiniBand fabric switch management interface access controls, RoCE fabric segmentation, ConnectX adapter (HDR/NDR) security settings, and subnet manager authentication and logging.
- Owner Primary: Network Engineering
- Owner Supporting: GPU Infrastructure Engineering; ML Platform Engineering; Security Engineering
