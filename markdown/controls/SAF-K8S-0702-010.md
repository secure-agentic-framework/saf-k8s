# SAF-K8S-0702-010 - Virtual cluster deployment for high-isolation multi-tenant Kubernetes environments

- Domain: D07 - Storage, Multi-tenancy, and Resource Governance
- Knowledge Area: 7.2 - Namespace Isolation and Multi-Tenancy
- Maturity: Advanced
- Classification: baseline

## Statement

Organizations requiring stronger tenant isolation than namespace-level controls provide must evaluate and implement virtual cluster technologies or equivalent tenant control-plane separation mechanisms to achieve API-server-level isolation between tenants while sharing underlying infrastructure.

## Objective

Provide stronger tenant isolation than namespace boundaries alone by deploying virtual cluster or equivalent control-plane separation for high-isolation tenants on shared infrastructure.

## Rationale

Namespace isolation has inherent limitations because cluster-scoped resources and host control-plane surfaces remain shared. Virtual clusters or equivalent tenant control-plane separation reduce the blast radius of tenant misconfiguration, privileged tenant actions, and cross-tenant interference in shared Kubernetes environments.

## Scope

Applies to multi-tenant Kubernetes environments where namespace-level isolation is insufficient for the threat model, including external tenants, cross-business-unit sharing, or mixed data classification levels. Not required for single-tenant or dedicated-cluster deployments.

## Enforcement

- Primary Enforcement Point: Virtual cluster provisioning system, host cluster admission controls restricting tenant access to the host API server, and tenant isolation architecture review.
- Implementation Surface: vCluster or equivalent virtual control-plane deployments, host cluster admission controls, dedicated control-plane separation patterns for high-isolation tenants, provisioning workflows for tenant virtual clusters, and compensating controls for shared-kernel or shared-node residual exposure.
- Owner Primary: Platform Engineering
- Owner Supporting: Security Engineering; Cloud Platform Engineering
