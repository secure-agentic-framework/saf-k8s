# SAF-K8S-0701-002 - CSI driver security and privilege restriction

- Domain: D07 - Storage, Multi-tenancy, and Resource Governance
- Knowledge Area: 7.1 - Persistent Storage Security
- Maturity: Practitioner
- Classification: baseline

## Statement

Organizations must restrict CSI driver deployments to approved and verified drivers, enforce least-privilege for driver service accounts, apply RBAC controls on volume snapshot operations, and restrict StorageClass creation to authorized administrators.

## Objective

Limit the privilege footprint of storage drivers and prevent unauthorized provisioning or snapshot access through least-privilege CSI driver configuration.

## Rationale

CSI drivers operate with elevated cluster privileges and can access raw block devices and host filesystems. An unauthorized or misconfigured driver can exfiltrate data, escalate privileges, or bypass namespace isolation. Unrestricted snapshot access allows unauthorized cloning of sensitive volumes.

## Scope

Applies to all Kubernetes clusters running CSI drivers. Covers driver installation, service account permissions, snapshot controller RBAC, and StorageClass management. Managed Kubernetes services must apply equivalent restrictions through provider-supported storage configuration.

## Enforcement

- Primary Enforcement Point: CSI driver deployment manifests, ClusterRole and ClusterRoleBinding for driver service accounts, VolumeSnapshotClass RBAC, StorageClass creation policies, and admission control rules restricting privileged CSI pods.
- Implementation Surface: CSIDriver and StorageClass Kubernetes resource types, CSI driver DaemonSet deployments and their ServiceAccount RBAC bindings, VolumeSnapshot/VolumeSnapshotContent/VolumeSnapshotClass CRDs, admission control policies blocking unapproved CSIDriver installation, AppArmor/SELinux profiles for CSI node pods, and runtime security tooling (Falco, Tetragon) monitoring CSI driver behavior.
- Owner Primary: Platform Engineering
- Owner Supporting: Storage Engineering; Security Engineering
