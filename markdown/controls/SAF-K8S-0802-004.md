# SAF-K8S-0802-004 - GPU firmware integrity monitoring

- Domain: D08 - GPU, Accelerator, and Confidential Computing
- Knowledge Area: 8.2 - GPU Driver, Library, and Toolkit Security
- Maturity: Advanced
- Classification: ai-specific

## Statement

GPU firmware integrity must be monitored for unauthorized modifications, firmware version changes outside authorized upgrade windows must be detected and alerted, and hardware anomaly monitoring must identify indicators of firmware-level tampering.

## Objective

Detect unauthorized firmware modifications on GPU hardware that could compromise the confidentiality and integrity of all AI workloads running on the affected device.

## Rationale

GPU firmware operates below the driver and OS layers. A compromised firmware image can intercept all data processed on the GPU, including model weights, training data, and inference inputs, while remaining invisible to OS-level security tools. Without firmware integrity monitoring, an attacker with one-time physical or privileged access can establish persistent GPU-level compromise that survives driver reinstallation and node reimaging.

## Scope

Applies to all Kubernetes GPU nodes with firmware that supports integrity verification (NVIDIA GPUs with InfoROM, firmware signing). Covers GPU firmware version tracking, firmware checksum validation, and hardware anomaly detection. Nodes using GPUs without firmware integrity features must document compensating controls.

## Enforcement

- Primary Enforcement Point: Node-level firmware integrity checking tools (nvidia-smi, InfoROM verification), DCGM hardware anomaly monitoring, and authorized firmware upgrade window controls.
- Implementation Surface: GPU firmware version tracking via nvidia-smi -q, NVIDIA InfoROM firmware integrity verification, DCGM hardware anomaly monitoring (ECC error patterns, clock throttling, power consumption), GPU firmware version and checksum inventory management, and authorized firmware upgrade window controls.
- Owner Primary: GPU Infrastructure Engineering
- Owner Supporting: Security Engineering
